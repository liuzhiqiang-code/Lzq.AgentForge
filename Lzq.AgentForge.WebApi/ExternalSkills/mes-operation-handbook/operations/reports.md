# 报表查询操作手册

## 功能概述

报表查询模块用于生成和查看各类生产报表，包括生产日报、质量统计、设备运行、人员绩效等。管理层可通过报表了解生产状况、分析问题和辅助决策。

## 操作入口

- **菜单路径**：报表中心 → 生产报表 / 质量报表 / 设备报表 / 绩效报表
- **权限要求**：生产经理（全部报表）、质量经理（质量报表）、厂长（全部报表）、计划员（生产报表）
- **快捷入口**：工作台 → 常用报表 / 我的订阅

## 操作步骤

### 场景1：生成生产日报

**适用角色**：生产经理、计划员、车间主任

**操作目的**：生成当日/昨日的生产完成情况报表

**操作步骤**：

1. 进入「报表中心 → 生产报表 → 生产日报」
2. 设置查询条件：
   - **日期**：选择要查询的日期（默认今天）
   - **产线**：全部产线 / 选择特定产线
   - **班次**：全部 / 白班 / 夜班
   - **产品**：全部 / 选择特定产品
3. 点击「生成报表」，系统显示：
   - **生产概览**：
     - 计划工单数、完成工单数
     - 计划产量、实际产量、达成率
     - 整体进度百分比
   - **产线明细**：
     - 各产线产量、工时、效率
     - 合格率、废品率
     - 异常停机时间
   - **工单明细**：
     - 每个工单的完成情况
     - 进度百分比、剩余数量
     - 操作工、设备信息
4. 可导出 Excel/PDF 格式，或发送到邮箱

**报表字段说明**：

| 字段 | 说明 | 计算公式 |
|------|------|---------|
| 计划产量 | 工单计划生产数量 | 工单创建时设定 |
| 实际产量 | 累计完成的好品数量 | 从报工汇总 |
| 达成率 | 产量完成百分比 | 实际产量 / 计划产量 × 100% |
| 工时效率 | 单位工时产量 | 实际产量 / 实作工时 |
| 废品率 | 废品占比 | 废品数量 / (实际产量+废品) × 100% |

**注意事项**：
- 💡 日报数据有 1 小时延迟（用于数据汇总和校正）
- 💡 可设置"每日自动生成"并发送到指定邮箱
- ⚠️ 如发现数据异常，联系计划员核实报工记录

### 场景2：生成质量统计报表

**适用角色****：质量经理、质量工程师、生产经理

**操作目的**：分析质量趋势，找出质量问题和改进方向

**操作步骤**：

1. 进入「报表中心 → 质量报表 → 质量统计报表」
2. 设置查询条件：
   - **时间范围**：查询起始和结束日期
   - **产品**：全部 / 选择特定产品
   - **工序**：全部 / 选择特定工序
   - **检验类型**：IQC / IPQC / FQC / 全部
3. 点击「生成报表」，系统显示：
   - **质量概览**：
     - 检验批次总数、合格批次、不合格批次
     - 整体合格率、不良品数量
     - 主要缺陷类型分布（柏拉图）
   - **趋势分析**：
     - 合格率趋势图（按日/周/月）
     - 不良率趋势图
     - 主要缺陷变化趋势
   - **详细数据**：
     - 各产品/工序的合格率对比
     - 供应商质量排名（IQC数据）
     - 检验员工作统计
4. 可导出 Excel/PDF，或生成 PPT 汇报材料

**质量统计关键指标**：

| 指标 | 计算公式 | 目标值 |
|------|---------|--------|
| 批次合格率 | 合格批次 / 总检验批次 × 100% | ≥ 98% |
| 产品直通率 | 一次性合格产品 / 总产量 × 100% | ≥ 95% |
| 不良品率 | 不良品数量 / 总产量 × 100% | ≤ 2% |
| 客户投诉率 | 客户投诉批次 / 发货批次 × 1000‰ | ≤ 5‰ |

**注意事项**：
- 💡 可点击柏拉图中的缺陷类型，查看该缺陷的详细分析
- 💡 趋势图支持按日/周/月切换，观察不同时间维度的变化
- ⚠️ 发现质量异常趋势，需立即启动改进措施

### 场景3：生成设备运行报表

**适用角色**：生产经理、设备主管、车间主任

**操作目的**：分析设备运行效率，找出OEE低的原因

**操作步骤**：

1. 进入「报表中心 → 设备报表 → 设备运行分析」
2. 设置查询条件：
   - **设备范围**：全部设备 / 设备组 / 单台设备
   - **时间范围**：查询起始和结束日期
   - **班次**：全部 / 白班 / 夜班
3. 点击「生成报表」，系统显示：
   - **OEE 概览**：
     - 平均OEE、最高/最低OEE设备
     - 时间开动率、性能开动率、合格品率
   - **OEE 损失分析**：
     - 停机损失饼图（故障/换模/待料/其他）
     - 性能损失分析（空转/降速）
     - 质量损失分析（报废/返工）
   - **设备明细**：
     - 各设备OEE、开机率、故障次数
     - 平均故障间隔时间（MTBF）
     - 平均修复时间（MTTR）
4. 可导出 Excel/PDF，或生成设备效率看板

**OEE 计算公式**：
```
OEE = 时间开动率 × 性能开动率 × 合格品率

时间开动率 = 实际运行时间 / 计划运行时间 × 100%
性能开动率 = 实际产量 / 理论产量 × 100%
合格品率 = 合格品数量 / 总产量 × 100%
```

**OEE 评价标准**：

| OEE 范围 | 评价 | 改进建议 |
|---------|------|---------|
| ≥ 85% | 世界级 | 保持，持续优化 |
| 75% - 85% | 良好 | 局部改进，缩小差距 |
| 65% - 75% | 一般 | 重点改进，找出瓶颈 |
| < 65% | 较差 | 全面诊断，系统改进 |

### 场景4：生成人员绩效报表

**适用角色**：生产经理、车间主任、HR

**操作目的**：统计操作工的生产效率、质量表现，用于绩效考核

**操作步骤**：

1. 进入「报表中心 → 绩效报表 → 人员绩效统计」
2. 设置查询条件：
   - **时间范围**：查询起始和结束日期
   - **人员范围**：全部 / 车间 / 班组 / 个人
   - **产线**：全部 / 选择特定产线
3. 点击「生成报表」，系统显示：
   - **绩效概览**：
     - 总人数、平均效率、平均合格率
     - 排名前三和后三的员工
   - **个人明细**：
     - 每个员工的产量、工时、效率
     - 合格率、废品率
     - 完成的工单数、设备使用时间
   - **绩效对比**：
     - 个人 vs 团队平均
     - 本月 vs 上月
     - 白班 vs 夜班
4. 可导出 Excel 用于绩效考核，或生成激励方案

**绩效评价指标**：

| 指标 | 权重 | 计算公式 | 目标值 |
|------|------|---------|--------|
| 产量完成率 | 40% | 实际产量 / 计划产量 × 100% | ≥ 100% |
| 工时效率 | 30% | 实际产量 / 实作工时 | 按标准工时 |
| 质量合格率 | 20% | 合格品 / 总产量 × 100% | ≥ 98% |
| 出勤率 | 10% | 实际出勤天数 / 应出勤天数 × 100% | ≥ 95% |

**注意事项**：
- 💡 绩效数据每月 5 号前自动锁定，不可修改上月数据
- 💡 可设置绩效预警，低于目标值时自动通知员工和主管
- ⚠️ 绩效数据涉及员工利益，务必准确公正

## 常见问题 (FAQ)

**Q: 报表数据与实际不符，怎么办？**
A: 请按以下步骤排查：
1. 确认查询条件是否设置正确（日期范围、产线等）
2. 检查是否有未审核的报工/检验记录
3. 联系计划员/质检员核实原始数据
4. 如确属系统错误，联系IT修复数据

**Q: 能自定义报表格式吗？**
A: 目前支持有限的自定义：
1. 可调整显示字段（显示/隐藏）
2. 可调整排序方式（按某个字段升序/降序）
3. 可保存常用查询条件为"我的视图"
4. 如需完全自定义报表，需联系IT开发

**Q: 如何设置报表自动推送？**
A: 设置报表订阅：
1. 生成报表后，点击「订阅此报表」
2. 选择推送频率：每日 / 每周 / 每月
3. 选择推送时间（如每日 8:00）
4. 填写接收邮箱（可多个，逗号分隔）
5. 系统将按时自动生成并发送报表

**Q: 历史报表数据能查询多久前的？**
A: 报表数据保留政策：
- ✅ 近 1 年数据：可在线查询，秒级响应
- ✅ 1-3 年数据：可查询，响应时间 1-5 分钟
- ⚠️ 3 年以上数据：需申请从归档库恢复，预计 1 个工作日
- 💡 建议重要报表定期导出备份

## 业务规则

### 报表生成时间规则

| 报表类型 | 数据更新频率 | 生成耗时 | 建议生成时间 |
|---------|-------------|---------|-------------|
| 生产日报 | 每小时 | < 10 秒 | 每日 8:00（自动） |
| 质量统计 | 实时 | 10-30 秒 | 每周一 9:00 |
| 设备运行 | 每小时 | 30-60 秒 | 每月 1 号 |
| 人员绩效 | 每日 | 1-3 分钟 | 每月 5 号 |

### 报表导出格式对比

| 格式 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| Excel (.xlsx) | 数据分析、二次加工 | 可编辑、支持公式 | 文件较大 |
| PDF | 正式汇报、打印 | 格式固定、美观 | 不可编辑 |
| CSV | 数据导入、系统对接 | 体积小、通用 | 无格式 |
| PPT | 会议汇报 | 图文并茂 | 需额外制作 |

### 报表权限矩阵

| 报表类型 | 操作工 | 班组长 | 车间主任 | 生产经理 | 厂长 |
|---------|--------|--------|---------|---------|------|
| 生产日报 | 本人数据 | 班组数据 | 车间数据 | ✅ 全部 | ✅ 全部 |
| 质量统计 | ❌ | 班组数据 | 车间数据 | ✅ 全部 | ✅ 全部 |
| 设备运行 | ❌ | 设备状态 | 车间设备 | ✅ 全部 | ✅ 全部 |
| 人员绩效 | 本人数据 | 班组数据 | 车间数据 | ✅ 全部 | ✅ 全部 |
