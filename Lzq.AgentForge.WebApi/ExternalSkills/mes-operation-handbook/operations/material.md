# 物料管理操作手册

## 功能概述

物料管理模块用于管理生产所需原材料、半成品、辅料的库存查询、领料、退料、入库等操作。仓库管理员、物料员和操作工日常工作中需使用此模块。

## 操作入口

- **菜单路径**：仓库管理 → 物料管理 / 库存查询
- **权限要求**：仓库管理员（全部权限）、物料员（领料退料）、操作工（查询+领料）
- **快捷入口**：工作台 → 库存预警 / 我的领料单

## 操作步骤

### 场景1：查询物料库存

**适用角色**：仓库管理员、物料员、计划员、操作工

**操作目的**：查询物料当前库存数量、库位、批次信息

**操作步骤**：

1. 进入「仓库管理 → 库存查询」页面
2. 输入查询条件：
   - **物料编码**：支持模糊查询，如输入"ABS"查询所有ABS材料
   - **物料名称**：按名称关键字查询
   - **物料类别**：原料 / 半成品 / 辅料 / 包材
   - **库存状态**：全部 / 正常 / 低于安全库存 / 零库存
3. 点击「查询」，查看库存列表：
   - 物料编码、名称、规格
   - 当前库存数量、单位
   - 安全库存预警线
   - 存放库位
   - 最新入库时间
4. 点击具体物料，查看：
   - 批次明细（批次号、数量、入库时间、供应商）
   - 库存变动记录
   - 库存趋势图

**库存状态标识**：
- 🟢 **正常**：库存 ≥ 安全库存
- 🟡 **预警**：库存 < 安全库存但 > 0
- 🔴 **缺货**：库存 = 0 或 < 最小订货点

**注意事项**：
- ⚠️ 库存数据实时更新，但存在 1-2 分钟延迟
- 💡 可导出库存清单用于盘点或采购计划
- 💡 点击库位编码可查看库位详情和存放规则

---

### 场景2：生产领料

**适用角色**：操作工、物料员

**操作目的**：根据工单需求领取生产所需物料

**操作步骤**：

1. 进入「仓库管理 → 领料管理 → 新建领料单」
2. 选择领料类型：
   - **生产领料**：根据工单BOM领取物料
   - **补料**：工单执行中补充物料
   - **试验领料**：用于新产品试制
3. 关联工单（生产领料必填）：
   - 输入或扫描工单号
   - 系统自动带出该工单的BOM清单
4. 填写领料明细：
   - 物料编码（可扫码）
   - 领料数量（默认按BOM需求量，可调整）
   - 批次选择（系统推荐先进先出批次）
   - 领用用途说明
5. 确认领料信息无误后，点击「提交领料单」
6. 仓库管理员审核后，凭领料单到仓库领料
7. 仓库发料时扫码确认，系统自动扣减库存

**业务规则**：
- 📌 生产领料数量 ≤ 工单需求数量 × (1 + 损耗率)
- 📌 系统按"先进先出"原则推荐批次
- 📌 领料单提交后需在 24 小时内完成领料，逾期自动取消
- 📌 特殊物料（危险品、贵重品）需主管审批

**注意事项**：
- ⚠️ 领料前确认工单状态为"processing"（进行中）
- ⚠️ 实际领料数量应与领料单一致，如有差异需备注说明
- 💡 可通过移动端扫码快速完成领料确认

### 场景3：生产退料

**适用角色**：操作工、物料员

**操作目的**：将工单剩余物料或不良物料退回仓库

**操作步骤**：

1. 进入「仓库管理 → 退料管理 → 新建退料单」
2. 选择退料类型：
   - **余料退库**：工单完工后的剩余好料
   - **不良退库**：质量不合格的物料
   - **代用退库**：规格不符需更换的物料
3. 关联工单（必填）
4. 填写退料明细：
   - 物料编码
   - 退料数量
   - 批次号（必填，需与原领料批次一致）
   - 退料原因
   - 不良说明（不良退料必填）
5. 上传退料照片（不良退料必填）
6. 提交退料单，仓库管理员审核
7. 实物退库，仓库扫码确认入库

**退料原因说明**：

| 退料类型 | 常见原因 | 处理方式 |
|---------|---------|---------|
| 余料退库 | 工单完工剩余 | 入库后可再次使用 |
| 不良退库 | 质量不合格 | 隔离，退回供应商 |
| 代用退库 | 规格不符 | 入库后重新领用正确规格 |

**注意事项**：
- ⚠️ 退料必须在工单关闭前完成
- ⚠️ 不良退料需质检员确认后方可退库
- ⚠️ 退料批次必须与原领料批次一致，禁止混批

### 场景4：查询安全库存预警

**适用角色**：仓库管理员、计划员、采购员

**操作目的**：及时发现库存不足的物料，安排采购或生产

**操作步骤**：

1. 进入「仓库管理 → 库存预警」页面
2. 系统自动列出所有预警物料：
   - 🔴 **缺货**：当前库存 = 0
   - 🟡 **低于安全库存**：当前库存 < 安全库存
   - 🟠 **即将缺货**：按当前消耗速度，X天内将缺货
3. 点击具体物料可查看：
   - 安全库存设置值
   - 日均消耗量
   - 预计可用天数
   - 在途采购订单（如有）
4. 可一键生成采购申请（需采购权限）
5. 或通知计划员调整生产计划

**预警处理优先级**：

| 预警级别 | 库存状态 | 处理时限 |
|---------|---------|---------|
| 🔴 紧急 | 已缺货 | 立即处理 |
| 🟡 重要 | 低于安全库存 | 24小时内 |
| 🟠 关注 | 即将缺货 | 3天内 |

**注意事项**：
- 💡 系统每日早上 8:00 自动发送预警邮件给仓库管理员和计划员
- 💡 可在物料主数据中调整安全库存设置（需主管审批）

## 常见问题 (FAQ)

**Q: 领料时发现库存不足怎么办？**
A: 请按以下步骤处理：
1. 联系计划员确认是否可调整工单数量
2. 联系采购员紧急采购（如为关键物料）
3. 查看是否有替代物料可用（需工艺工程师确认）
4. 如无法解决，需暂停工单等待物料

**Q: 领料后发现物料不良，能直接退库吗？**
A: 不可以。必须先：
1. 通知质检员进行来料检验（IQC）
2. 质检员确认不良后出具不合格报告
3. 凭不合格报告办理不良退库
4. 重新领用合格物料

**Q: 如何查询某个批次物料的去向？**
A: 使用追溯查询功能：
1. 进入「追溯管理 → 反向追溯」
2. 输入物料批次号
3. 系统显示该批次物料被用于哪些工单、哪些成品
4. 可进一步追溯至最终客户

**Q: 领料单提交后想修改，怎么办？**
A: 领料单在"待审核"状态前可自行撤销：
1. 进入「领料管理 → 我的领料单」
2. 找到对应领料单，点击「撤销」
3. 重新创建正确的领料单
4. 如已审核通过，需联系仓库管理员作废

## 业务规则

### 物料编码规则

| 物料类别 | 编码前缀 | 示例 |
|---------|---------|------|
| 原材料 | RM | RM-ABS-001 |
| 半成品 | SF | SF-PART-001 |
| 辅料 | AU | AU-OIL-001 |
| 包材 | PK | PK-BOX-001 |
| 成品 | FG | FG-PROD-001 |

### 库存状态定义

| 状态 | 定义 | 颜色标识 |
|------|------|---------|
| 正常 | 库存 ≥ 安全库存 | 🟢 绿色 |
| 预警 | 安全库存 > 库存 > 0 | 🟡 黄色 |
| 缺货 | 库存 = 0 | 🔴 红色 |
| 冻结 | 质检中/待处理 | ⚫ 灰色 |

### 领料审批规则

| 领料类型 | 金额/数量条件 | 审批要求 |
|---------|--------------|---------|
| 生产领料 | 常规物料 | 仓库管理员直接发料 |
| 生产领料 | 贵重物料（>5000元） | 需主管审批 |
| 补料 | 超出BOM用量 | 需计划员+主管审批 |
| 试验领料 | 任何金额 | 需工艺工程师审批 |
