# 工艺管理操作手册

## 功能概述

工艺管理模块用于维护产品的工艺路线、标准工时、BOM结构和工艺参数等信息。工艺工程师和计划员需使用此模块进行工艺设计和查询。

## 操作入口

- **菜单路径**：工艺管理 → 工艺路线 / BOM管理 / 标准工时
- **权限要求**：工艺工程师（维护+查询）、计划员（查询）、操作工（查询自己工单的工艺）
- **快捷入口**：工作台 → 我的工艺路线

## 操作步骤

### 场景1：查询产品工艺路线

**适用角色**：计划员、工艺工程师、操作工

**操作目的**：查看产品从原材料到成品的完整加工流程

**操作步骤**：

1. 进入「工艺管理 → 工艺路线查询」
2. 输入查询条件：
   - **产品编码**：输入或扫描产品编码
   - **产品名称**：按名称模糊查询
   - **工艺版本**：选择特定版本（如不填则显示最新版本）
3. 点击「查询」，查看工艺路线列表
4. 点击具体产品，查看完整工艺路线：
   - **工序顺序**：第1道工序 → 第2道工序 → ... → 完工
   - **工序名称**：如注塑、 CNC加工、打磨、涂装等
   - **加工设备**：每道工序使用的设备类型
   - **标准工时**：每道工序的标准作业时间
   - **工艺要求**：关键工艺参数和质量要求

**典型工艺路线示例**（注塑产品）：
```
原材料检验 → 注塑成型 → 去毛刺 → 检验 → 包装 → 入库
   ↓           ↓           ↓        ↓       ↓
 IQC检验   注塑机A-01   人工/设备   IPQC   包装台
 0.5h       2h          0.5h      0.5h    0.5h
```

**注意事项**：
- 💡 工艺路线版本号格式：`V1.0`、`V2.0`，数字越大版本越新
- 💡 可点击工序查看详细工艺指导书（PDF）
- ⚠️ 工艺路线变更需走变更流程，不可随意修改

### 场景2：查看标准工时

**适用角色**：工艺工程师、计划员、成本会计

**操作目的**：查询各工序的标准作业时间，用于生产计划和成本核算

**操作步骤**：

1. 进入「工艺管理 → 标准工时查询」
2. 选择查询方式：
   - **按产品查询**：查看某产品的全部工序工时
   - **按工序查询**：查看某工序在不同产品中的工时
3. 输入查询条件并点击「查询」
4. 查看标准工时列表：
   - 产品编码、名称
   - 工序名称、工序序号
   - **准备时间**：换模、调机等准备工作的标准时间
   - **单件工时**：生产一件产品的标准时间
   - **标准产能**：每小时标准产量（60分钟/单件工时）
5. 可导出 Excel 用于成本分析和报价

**标准工时构成**：
```
标准工时 = 准备时间 + (单件工时 × 生产数量)
单件工时 = 加工时间 + 辅助时间 + 宽放时间
```

**宽放时间说明**：
- 📌 疲劳宽放：5-10%（重体力工作取高值）
- 📌 私事宽放：2-5%（上厕所、喝水等）
- 📌 管理宽放：3-5%（等待物料、设备等）

**注意事项**：
- ⚠️ 标准工时用于生产计划和绩效核算，必须准确
- 💡 新工艺投入前应通过节拍测定验证标准工时
- 💡 每隔6个月应重新测定标准工时，确保准确性

### 场景3：查看产品BOM（物料清单）

**适用角色**：计划员、物料员、工艺工程师

**操作目的**：查看产品所需的原材料、零部件清单和用量

**操作步骤**：

1. 进入「工艺管理 → BOM查询」
2. 输入产品编码或名称，点击「查询」
3. 系统显示该产品的BOM结构：
   - **层级**：0级（成品）→ 1级（部件）→ 2级（原材料）
   - **物料编码**：BOM中每个物料的唯一编码
   - **物料名称**：物料名称、规格型号
   - **单位用量**：生产一件产品所需的数量
   - **损耗率**：合理损耗比例（如注塑3%）
   - **净需求** = 计划产量 × 单位用量 × (1 + 损耗率)
4. 点击「展开BOM」可查看多层BOM结构
5. 点击「BOM对比」可对比不同版本的BOM差异

**BOM层级示例**：
```
0级：成品 P-001 手机壳
  ├─1级：原料 RM-ABS-001 ABS塑料粒子（0.05kg，损耗3%）
  ├─1级：辅料 AU-PAINT-001 喷涂油漆（0.01kg，损耗5%）
  └─1级：包材 PK-BOX-001 包装盒（1个，损耗0%）
```

**注意事项**：
- ⚠️ BOM是物料需求计算的基准，必须准确维护
- 💡 工程中BOM（EBOM）和制造BOM（MBOM）可能不同，注意区分
- ⚠️ BOM变更需通知计划员、采购员和仓库管理员

### 场景4：查询工艺参数

**适用角色**：操作工、工艺工程师、设备维修工

**操作目的**：查看特定工序的关键工艺参数设置范围

**操作步骤**：

1. 进入「工艺管理 → 工艺参数查询」
2. 选择查询条件：
   - **产品编码**：选择特定产品
   - **工序名称**：选择具体工序（如注塑、CNC等）
   - **设备类型**：选择设备类型（可选）
3. 点击「查询」，查看工艺参数表：
   - **参数名称**：如温度、压力、转速、时间等
   - **标准值**：工艺要求的标准设定值
   - **上限**：参数允许的最大值
   - **下限**：参数允许的最小值
   - **控制方式**：人工设定 / 自动闭环控制
4. 点击「查看SOP」可下载该工序的标准作业指导书

**注塑工序典型工艺参数**：
| 参数名称 | 标准值 | 下限 | 上限 | 单位 |
|---------|--------|------|------|------|
| 料筒温度 | 220 | 210 | 230 | ℃ |
| 模具温度 | 60 | 55 | 65 | ℃ |
| 注射压力 | 80 | 70 | 90 | bar |
| 保压时间 | 15 | 12 | 18 | s |
| 冷却时间 | 30 | 25 | 35 | s |

**注意事项**：
- ⚠️ 工艺参数直接影响产品质量，必须严格按标准设定
- ⚠️ 参数调整需工艺工程师批准，操作工不可擅自修改
- 💡 工艺参数异常会自动报警并记录，用于质量追溯

## 常见问题 (FAQ)

**Q: 查询工艺路线时提示"未找到该产品的工艺路线"，怎么办？**
A: 可能原因及处理方式：
1. 产品编码输入错误：检查编码是否正确（区分大小写）
2. 该产品尚未建立工艺路线：联系工艺工程师创建
3. 工艺路线已作废：查看历史版本，或联系工艺工程师确认

**Q: 标准工时和实际工时有很大差异，怎么办？**
A: 请按以下步骤处理：
1. 确认操作是否符合标准作业程序（SOP）
2. 检查设备是否正常运行，无异常停机
3. 联系工艺工程师重新测定标准工时
4. 如确实需要调整，提交《标准工时变更申请》

**Q: BOM中某个物料要替换，如何操作？**
A: BOM变更需走正式流程：
1. 工艺工程师在系统中创建新的BOM版本
2. 填写《工程变更通知单》（ECN）
3. 经研发、质量、采购、生产会签批准
4. 新版本生效后，旧版本自动失效
5. 库存的老物料按处置方案使用或报废

**Q: 工艺参数可以在设备上直接修改吗？**
A: 不可以。工艺参数修改必须：
1. 操作工发现问题，报告班组长或工艺工程师
2. 工艺工程师现场确认，填写《工艺参数调整单》
3. 经质量工程师会签确认不影响质量
4. 在系统中更新工艺参数标准值
5. 新参数生效后方可按新标准生产

## 业务规则

### 工艺路线版本管理

| 版本类型 | 版本号 | 说明 |
|---------|--------|------|
| 试产版本 | V0.x | 试生产阶段，工艺不稳定 |
| 正式版本 | V1.0+ | 量产阶段，工艺稳定 |
| 变更版本 | Vx.y | 工程变更后的新版本 |

- 📌 工艺路线版本变更需填写《工程变更通知单》（ECN）
- 📌 新旧版本切换需确保库存物料、在制品的处置方案
- 📌 旧版本需保留至少 2 年，用于追溯和历史数据分析

### BOM准确率要求

- 📌 BOM准确率目标：≥ 99.5%
- 📌 每月抽查 10% 的BOM进行准确性验证
- 📌 BOM错误导致的问题需追责到相关工程师

### 标准工时更新周期

| 工序类型 | 更新周期 | 触发条件 |
|---------|---------|---------|
| 新工序 | 3个月后 | 量产稳定后首次更新 |
| 成熟工序 | 6个月 | 定期复核 |
| 异常工序 | 立即 | 实际工时持续超差 10%+ |
