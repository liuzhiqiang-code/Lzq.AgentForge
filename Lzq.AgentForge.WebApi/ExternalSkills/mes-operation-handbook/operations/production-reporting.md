# 生产报工操作手册

## 功能概述

生产报工模块用于操作工和班组长记录生产过程中的产量、工时、废品等数据。准确的报工数据是生产统计、成本核算、绩效分析的基础。

## 操作入口

- **菜单路径**：生产管理 → 生产报工
- **权限要求**：操作工（报工自己工单）、班组长（报工班组工单）、计划员（查询+修正）
- **快捷入口**：工作台 → 待报工工单 / 移动端扫码报工

## 操作步骤

### 场景1：日常生产报工

**适用角色**：操作工、班组长

**操作目的**：记录当班完成的产品数量、工时、废品等信息

**操作步骤**：

1. 进入「生产管理 → 生产报工」页面
2. 在"待报工工单"列表中找到当前工单，点击「报工」
3. 填写报工信息：
   - **完成数量**：本次实际完成的好品数量（必填）
   - **废品数量**：本次产生的废品数量（如有）
   - **工废数量**：因操作失误导致的废品
   - **料废数量**：因材料质量问题导致的废品
   - **实作工时**：本次操作实际花费的工时（小时，必填）
   - **人数**：参与本次作业的人数（默认1人）
   - **备注**：填写异常说明或特殊情况（可选）
4. 确认信息无误后，点击「提交报工」
5. 系统自动：
   - 更新工单累计完成数量和进度百分比
   - 扣减对应物料库存
   - 记录操作人和操作时间

**业务规则**：
- 📌 完成数量 + 废品数量 ≤ 工单剩余数量
- 📌 实作工时 > 0，且 ≤ 24 小时（单班次上限）
- 📌 废品需区分工废和料废，影响责任判定和成本核算
- 📌 同一工单每班次最多报工 5 次（防误操作）

**注意事项**：
- ⚠️ 报工数据直接影响生产统计和成本核算，务必准确填写
- ⚠️ 报工后发现错误需立即联系计划员修正，不可自行删除
- 💡 建议每班次结束前完成报工，避免数据积压和遗忘
- 💡 可通过移动端扫码快速报工，无需登录电脑

### 场景2：异常情况报工

**适用角色**：操作工、班组长

**操作目的**：生产过程中出现设备故障、质量问题等异常时的报工处理

**操作步骤**：

1. **设备故障导致停工**：
   - 填写报工前先填写「异常记录」
   - 选择异常类型：设备故障
   - 填写停机开始时间和结束时间
   - 在报工中扣除故障停机时间
   
2. **质量问题导致返工**：
   - 返工品不计入完成数量
   - 在"废品数量"中填写返工品数量
   - 在备注中注明"返工"
   - 返工完成后单独报工

3. **物料短缺中途换单**：
   - 先报工当前工单的完成数量
   - 将剩余物料退库（参见解料操作手册）
   - 新建另一工单的领料和报工

**异常类型代码**：

| 异常类型 | 代码 | 处理方式 |
|---------|------|---------|
| 设备故障 | EQ-001 | 报修+停机工时扣除 |
| 物料短缺 | MT-001 | 暂停工单+补料/换单 |
| 质量问题 | QT-001 | 返工/报废+原因分析 |
| 工艺调整 | PR-001 | 暂停+等待工艺确认 |
| 人员不足 | HR-001 | 调整排班+延期 |

**注意事项**：
- ⚠️ 异常报工必须填写详细备注，说明异常情况
- ⚠️ 异常导致的停工工时不得计入实作工时
- 💡 重大异常（停机>2小时）需立即上报车间主任

### 场景3：查询历史报工记录

**适用角色**：操作工、班组长、计划员

**操作目的**：查询自己和班组的报工历史，核对数据

**操作步骤**：

1. 进入「生产管理 → 生产报工 → 报工记录查询」
2. 设置查询条件：
   - **工单号**：查询特定工单的报工记录
   - **操作人**：查看自己或他人的报工记录
   - **日期范围**：查询某时间段的报工
   - **产线/设备**：按产线或设备筛选
3. 点击「查询」，查看报工记录列表：
   - 报工时间、操作人
   - 完成数量、废品数量
   - 实作工时、工时效率
   - 备注信息
4. 点击具体记录可查看详情和关联信息
5. 可导出 Excel 进行统计分析

**报工统计指标**：

| 指标 | 计算公式 | 说明 |
|------|---------|------|
| 工时效率 | 完成数量 / 实作工时 | 单位工时的产量 |
| 废品率 | 废品数量 / (完成数量+废品数量) × 100% | 废品占比 |
| 达成率 | 完成数量 / 计划数量 × 100% | 计划完成情况 |

### 场景4：移动端扫码报工

**适用角色**：操作工

**操作目的**：通过手机快速完成报工，无需登录电脑

**操作步骤**：

1. 打开企业微信/钉钉中的 MES 小程序
2. 点击「扫码报工」
3. 扫描工单二维码（工单看板上提供）
4. 系统自动带出工单信息：
   - 产品名称、计划数量
   - 已报工数量、剩余数量
5. 填写本次报工数据：
   - 完成数量（扫码后输入）
   - 废品数量（可选）
   - 实作工时（自动记录扫码至提交的时间）
6. 点击「提交」，完成报工

**注意事项**：
- 💡 移动端报工自动记录GPS位置和操作时间，防止虚假报工
- 💡 可上传现场照片（最多3张）作为报工凭证
- ⚠️ 移动端报工后仍需到电脑端核对数据准确性

## 常见问题 (FAQ)

**Q: 报工后发现数量填错了，能自己修改吗？**
A: 不可以。报工记录提交后：
1. 不可自行修改或删除
2. 需立即联系计划员说明情况
3. 计划员核实后可在后台修正
4. 重大错误需填写《报工数据更正申请单》

**Q: 多报了数量怎么办？**
A: 多报数量需在下一次报工中扣除（不能负数报工）：
1. 本次报工数量 = 实际完成 - 多报数量
2. 在备注中注明"扣除上次多报X件"
3. 联系计划员核实并备注说明
4. 确保累计数量准确

**Q: 报工时系统提示"超出剩余数量"，怎么办？**
A: 请按以下步骤处理：
1. 点击工单号查看"报工记录"tab
2. 核对累计完成数量 + 累计废品数量
3. 计算得出剩余数量 = 计划数量 - 累计总数
4. 按剩余数量报工，或联系计划员调整工单数量

**Q: 本班次只干了半小时就停机了，工时怎么填？**
A: 实作工时只填写实际生产的时间：
1. 停机时间不计入实作工时
2. 在备注中注明"设备XX:00-XX:00故障停机"
3. 如果是等待物料，注明"XX:00-XX:00待料"
4. 确保实作工时真实反映生产投入

**Q: 返工品需要报工吗？**
A: 返工分为两种情况：
1. **本工序返工**：不额外报工，返工完成后报一次合格品即可
2. **跨工序返工**：需单独建立返工工单，按新工单报工
3. 返工产生的废品需计入废品数量

## 业务规则

### 报工时间规则

- 📌 报工时间必须在工单开始时间之后、关闭时间之前
- 📌 允许补报最近 3 天的报工记录（超时需主管审批）
- 📌 系统按报工时间自动归入对应班次（白班/夜班）

### 数量校验规则

```
剩余数量 = 计划数量 - 累计完成数量 - 累计废品数量
本次报工数量 ≤ 剩余数量
累计完成数量 + 累计废品数量 ≤ 计划数量
```

### 工时计算规则

- 📌 实作工时：实际用于生产的时间（不含停机、待料、吃饭等）
- 📌 准实作工时：系统按"完成数量 / 标准工时"自动计算的理论工时
- 📌 工时效率 = 完成数量 / 实作工时（件/小时）

### 废品分类规则

| 废品类型 | 责任方 | 处理方式 |
|---------|--------|---------|
| 工废（操作失误） | 操作工 | 计入操作工绩效，需培训 |
| 料废（材料问题） | 供应商 | 退料+索赔，不扣操作工绩效 |
| 模废（模具问题） | 模具维修 | 维修模具，记录维修成本 |
| 设废（设备问题） | 设备维修 | 维修设备，记录停机损失 |
