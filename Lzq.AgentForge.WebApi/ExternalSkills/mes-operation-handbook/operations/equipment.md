# 设备管理操作手册

## 功能概述

设备管理模块用于监控车间设备的运行状态、查询设备信息、报修故障设备、查看设备保养记录等。设备维修工和车间主任需日常使用此模块。

## 操作入口

- **菜单路径**：设备管理 → 设备台账 / 设备状态监控
- **权限要求**：设备维修工（全部权限）、车间主任（查询+报修）、操作工（查询）
- **快捷入口**：工作台 → 设备状态看板

## 操作步骤

### 场景1：查询设备实时状态

**适用角色**：车间主任、操作工、设备维修工

**操作目的**：查看设备当前运行状态（运行/停机/故障/保养）

**操作步骤**：

1. 进入「设备管理 → 设备状态监控」页面
2. 查看设备状态看板，设备会按状态用不同颜色标识：
   - 🟢 **运行中（Running）**：设备正常生产
   - 🟡 **待机中（Idle）**：设备空闲待命
   - 🔴 **故障中（Fault）**：设备故障停机
   - 🔵 **保养中（Maintenance）**：计划保养中
   - ⚫ **离线（Offline）**：设备未连接
3. 点击具体设备卡片，查看详细信息：
   - 设备基本信息（编号、名称、型号）
   - 当前状态详情
   - 运行时长、停机次数等统计
   - 最近保养记录

**注意事项**：
- ⚠️ 状态数据每 30 秒刷新一次，若长时间未更新请联系IT检查网络连接
- 💡 可点击状态筛选按钮，快速过滤特定状态的设备
- 💡 设备编号格式：`设备类型-序号`，如 `INJ-001`（注塑机1号）

### 场景2：设备故障报修

**适用角色**：操作工、车间主任、设备维修工

**操作目的**：设备出现故障时，及时报修并通知维修人员

**操作步骤**：

1. 在设备状态看板找到故障设备，点击「报修」按钮
2. 填写报修信息：
   - **故障类型**：机械故障 / 电气故障 / 程序故障 / 其他
   - **故障描述**：详细描述故障现象（必填）
   - **严重程度**：
     - 🔴 **紧急**：设备完全停机，影响生产
     - 🟡 **一般**：设备部分功能异常，可降负荷运行
     - 🟢 **轻微**：小问题，待计划保养时处理
   - **报修人联系方式**：自动带出当前用户，可修改
3. 点击「提交报修」
4. 系统自动：
   - 将设备状态改为"故障中"
   - 生成维修工单并分配给维修工
   - 发送通知给相关维修人员

**业务规则**：
- 📌 紧急故障会在 5 分钟内通知到所有设备维修工
- 📌 报修后设备自动暂停，不可继续生产
- 📌 故障描述越详细，维修效率越高

**注意事项**：
- ⚠️ 紧急故障应先停机断电，确保现场安全后再报修
- ⚠️ 报修后请保持电话畅通，维修工可能需要现场确认
- 💡 可上传故障照片（最多3张），帮助维修工快速定位问题

### 场景3：查看设备保养计划

**适用角色**：设备维修工、车间主任

**操作目的**：查看设备的定期保养计划和执行记录

**操作步骤**：

1. 进入「设备管理 → 设备台账」页面
2. 搜索目标设备（可按设备编号、名称筛选）
3. 点击设备记录，选择「保养记录」标签页
4. 查看：
   - **保养计划**：下次保养日期、保养类型（日保/周保/月保/年保）
   - **历史记录**：过往保养时间、保养内容、执行人
   - **保养标准**：各类保养的检查项目和标准

**保养类型说明**：

| 保养类型 | 周期 | 主要内容 |
|---------|------|---------|
| 日保 | 每班 | 清洁、润滑、点检 |
| 周保 | 每周 | 紧固、调整、更换易损件 |
| 月保 | 每月 | 精度校验、性能测试 |
| 年保 | 每年 | 全面检修、大修 |

**注意事项**：
- 📌 保养到期前 3 天系统会自动提醒
- ⚠️ 保养期间设备不可用于生产，请提前安排生产计划
- 💡 保养完成后需填写保养记录，上传保养照片

### 场景4：查询设备OEE（设备综合效率）

**适用角色**：车间主任、生产经理

**操作目的**：分析设备运行效率，找出影响效率的原因

**操作步骤**：

1. 进入「设备管理 → 设备效率分析」页面
2. 设置查询条件：
   - **设备范围**：选择单台设备或设备组
   - **时间范围**：查询起始日期和结束日期
   - **班次**：全部 / 白班 / 夜班
3. 点击「查询」，查看 OEE 分析结果：
   - **OEE 总体值**：目标 ≥ 85%
   - **时间开动率**：设备实际运行时间 / 计划运行时间
   - **性能开动率**：实际产量 / 理论产量
   - **合格品率**：合格品数量 / 总产量
4. 点击「详情」可查看 OEE 损失分析饼图

**OEE 计算公式**：
```
OEE = 时间开动率 × 性能开动率 × 合格品率
```

**注意事项**：
- 💡 OEE 数据有 1 小时延迟，非实时数据
- 💡 可导出 Excel 报表用于分析汇报
- ⚠️ 若 OEE 长期低于 70%，需进行专项改善

## 常见问题 (FAQ)

**Q: 设备状态显示"离线"，但实际在运行，怎么办？**
A: 这是数据采集器通信中断导致。请：
1. 检查设备网络指示灯是否正常
2. 联系 IT 人员检查数据采集器状态
3. 临时使用人工记录，待恢复后补录数据

**Q: 报修后多久会有维修工来处理？**
A: 根据故障严重程度：
- 紧急故障：30 分钟内到场
- 一般故障：2 小时内到场
- 轻微故障：24 小时内安排

**Q: 设备保养期间可以临时用于生产吗？**
A: 不可以。保养期间设备：
1. 系统自动锁定，不可开工单
2. 擅自使用可能导致设备损坏或安全事故
3. 如确需紧急生产，需车间主任审批后暂停保养

**Q: OEE 数据从哪来？如何保证准确性？**
A: OEE 数据来源：
1. 运行时间：从设备 PLC 自动采集
2. 产量数据：从工单报工自动汇总
3. 合格品数据：从质量检验记录自动计算
4. 人工可校正异常数据（需计划员权限）

## 业务规则

### 设备状态流转

```
运行（Running） ⇄ 待机（Idle）
      ↓
   故障（Fault）→ 维修中（Repairing）→ 运行 / 待机
      ↓
   保养（Maintenance）→ 待机
      ↓
   离线（Offline）→ 运行（恢复通信后）
```

### 设备编号规则

| 设备类型 | 编号前缀 | 示例 |
|---------|---------|------|
| 注塑机 | INJ | INJ-001 |
| CNC机床 | CNC | CNC-002 |
| 冲压机 | PRS | PRS-001 |
| 烘干机 | DRN | DRN-003 |
| 机器人 | RBT | RBT-001 |

### 保养预警规则

- 🟢 正常：距离保养 > 7 天
- 🟡 预警：距离保养 ≤ 7 天
- 🔴 超期：已超过保养日期
